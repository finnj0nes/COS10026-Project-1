Enhancements

  1. Usemap in the About page
    • An Enhancement which we implemented is an HTML image map combined with CSS hover-over tooltips that identify each team member.
    • Enhancement is seen in the group photo in the About page here: about.html
    • Source used for technique: www.w3schools.com/html/html_images_imagemap.asp 

This enhancement goes past the lecture slides and tutorials by using the image map, adding to the user interactivity and giving instant access to information without requiring extra clicks. The feature has been tested in Firefox, as well as other browsers.

The code for this involves the group photo <img> element with a <usemap> attribute, linked to a <map> element containing multiple <area> elements - which are clickable regions in the image. Each region has a tooltip associated - using CSS, which appears when the user hovers over that <area>. This tooltip identifies the group member. In CSS, each tooltip is positioned relative to the <area> element rather than the <img> element.
